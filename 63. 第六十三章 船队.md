# 第六十三章 船队
# 
开铺子的第五年。
时蕴之终于买下了第一艘船。
不大。
旧船。
船身还有修补过的痕迹。
连名字都是别人留下的。
叫：
平安号。
掌柜们都不理解。
“东家。”
“为什么不买新的？”
时蕴之蹲在码头边。
摸了摸船身。
笑着说道：
“因为便宜。”
众人哭笑不得。
他们总觉得。
时蕴之不像个大东家。
该花钱的时候不花。
不该花钱的时候又很大方。
可他们慢慢发现。
她似乎很少冲动。
买东西之前。
总会先观察很久。
像下棋一样。
先看局势。
再落子。
平安号下水那天。
时蕴之站在码头看了很久。
心情有些奇怪。
以前。
她卖的是货。
现在。
她卖的是路。
因为船不是货物。
船是连接货物的东西。
这是完全不同的生意。
第一趟航程。
很顺利。
赚了钱。
第二趟。
也很顺利。
第三趟。
出事了。
暴雨。
江面涨水。
平安号被困在半路。
整整十天。
货物虽然保住了。
却错过了交货时间。
赔了一大笔违约银。
掌柜们急得团团转。
时蕴之倒是很平静。
因为她发现。
自己以前一直在和人打交道。
现在开始和天打交道了。
而天从来不讲道理。
那年冬天。
她亏了不少。
好不容易赚的钱。
又吐出去大半。
伙计们愁眉苦脸。
有人甚至偷偷劝她：
“要不卖船吧。”
时蕴之摇头。
“不卖。”
“为什么？”
她望着江面。
轻声说道：
“因为问题不在船。”
后来她整整研究了三个月。
看水文。
看地图。
看天气。
看历年洪水记录。
最后发现。
问题确实不在船。
而在路线。
于是第二年。
她换了一条航线。
虽然远一点。
却安全很多。
果然。
亏损立刻少了。
掌柜们这才服气。
渐渐地。
商行又赚回来了。
第三年。
买下第二艘船。
第四年。
第三艘。
第五年。
第五艘。
码头上开始有人叫她：
时掌柜。
也有人叫她：
时东家。
可时蕴之自己知道。
她依旧是个学生。
因为每一天都在学新的东西。
有一次。
一个老船工忽然问她：
“东家。”
“你知道船最重要的是什么吗？”
时蕴之想了想。
“结实？”
老船工摇头。
“方向。”
“方向？”
“嗯。”
老人指着江面。
“船漏了还能修。”
“帆坏了还能换。”
“可方向错了。”
“跑得越快。”
“死得越快。”
时蕴之愣住了。
那一刻。
她忽然觉得。
这话好像不只是说船。
夜里。
她把这句话记进《人间记》。
然后坐在窗边发呆。
这些年。
她学医。
学棋。
学瓷。
学刀。
学商。
每学一样东西。
最后都会变成一句关于人生的话。
仿佛所有道路走到尽头。
都在通往同一个地方。
窗外。
江风吹动灯笼。
远处商船缓缓驶过。
时蕴之忽然意识到。
自己已经不是那个在死人村里挣扎活下来的少女了。
她开始拥有影响别人的能力。
一艘船。
养活几十个人。
五艘船。
养活几百个人。
如果有一天。
是一百艘船呢？
她第一次认真思考这个问题。
不是因为贪婪。
而是因为责任。
因为船越多。
跟着她吃饭的人就越多。
而责任这种东西。
比赚钱重得多。
那天深夜。
她走进归屋。
长廊依旧安静。
棋室。
瓷室。
兵器室。
墨室。
酒室。
一间间房间沉默矗立。
她忽然发现。
归屋收藏的是过去。
而船队代表未来。
这是第一次。
她开始主动创造什么。
而不仅仅只是收藏什么。
